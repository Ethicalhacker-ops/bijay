# Jay Prasad Majhi - Personal Portfolio Website

Welcome to the official portfolio of **Jay Prasad Majhi**, a passionate IT professional with expertise in web development, consulting, and business operations. This website serves as a personal space where visitors can learn more about my background, skills, projects, and get in touch.

## Website Overview

This portfolio website is built to showcase my professional achievements, projects, skills, and connect with potential clients and collaborators. Below is a brief summary of the key sections:

### Key Features
1. **Home Section**  
   The homepage features an introduction to my portfolio and a live clock showing the current time.

2. **About Me Section**  
   This section provides details about my professional background, including my education, skills, and work experience. You can also find my personal photo here.

3. **Portfolio Section**  
   In this section, you can see the projects I have worked on, including my **Football News Website** and **Tech Consultancy**.

4. **Resume Section**  
   This section includes my educational qualifications, work experience, and notable achievements in the IT and business consulting industry.

5. **Blog Section**  
   The blog will contain articles and insights on web development, IT consulting, and business operations. Stay tuned for future updates.

6. **Testimonials Section**  
   Feedback and testimonials from clients, colleagues, and collaborators to showcase my work ethic and skills.

7. **Contact Section**  
   The contact form allows users to send messages directly. I’ve also included links to my social media profiles for easier communication.

8. **Quick Links**  
   At the top, you will find a navigation bar with quick links to all major sections, providing users with an easy way to navigate through the site.

9. **Dark Mode & Light Mode Toggle**  
   A toggle button allows users to switch between dark and light modes based on their preference.

## Technologies Used
- **HTML5**: For creating the basic structure of the website.
- **CSS3**: For styling the website and adding layout and design.
- **JavaScript**: For adding interactivity, like live clock and theme toggle.
- **GitHub Pages (optional)**: For hosting the website.
